CLI basic commands.

Type the commands shown below. Do not copy and paste. Type!



Exercise 1.   Find out your computer name.

hostname

echo %computername%



Exercise 2.   Find out the user account you're logged in as.

whoami

echo %username%



Exercise 3.  Find out the operating system you're running.

winver

echo %os%



Exercise 4.  Find out "a lot of info" about your system.

systeminfo

systeminfo > mysysteminfo.txt

mysysteminfo.txt

--- Now close the file.

msinfo32

dxdiag



Exercise 5.  Find out your processor architecture, identifier, level and revision.

echo %processor_architecture%

echo %processor_identifier%

echo %processor_level%

echo %processor_revision%



Exercise 6.  Basic folder and file creation.

cd \

mkdir Stuff

cd Stuff

echo Mike Meyers or James Messer? > both.txt

type both.txt

echo Study both, of course. >> both.txt

type both.txt

dir

notepad both.txt:topsecret.txt

--- click yes, type any message and close it.

dir

dir /a:h

notepad both.txt:topsecret.txt

--- close it. You just used ADS (alternate data streams, a way to add metadata to your files).



Exercise 7.  More ADS.

cd \

mkdir ADS

cd ADS

echo Having fun. > fun.txt

dir

echo wscript.echo "You were hacked!" > fun.txt:hacked.vbs

wscript fun.txt:hacked.vbs

--- You just ran a Visual Basic script that could have done a lot of damage ...



Exercise 8.  Creating and deleting files and folders.

cd \

mkdir Resistance.

cd Resistance

for /L %i in (1,1,9) do echo Libertad. > archive%i.txt

dir

for /L %i in (0,5,100) do echo Quiero cerveza. > file%i.txt

dir

mkdir Beer

cd Beer

echo Dos Equis > xx.txt

type xx.txt

dir

cd ..

dir /a:d

rd /q /s Beer

dir /a:d

dir

del *.txt

dir



Exercise 9.   Sequentially running commands.

cls & cd \ & hostname & whoami & title H@ck3r 2017.



Exercise 10.   Finding out running processes.

tasklist

tasklist | more

--- hit the spacebar to continue.

tasklist /svc

tasklist /?



Exercise 11.   An elegant way to find out the IPv4 address.

netsh interface ip show address
