CEHv9 exercises.


The virtual machines needed for the labs are shared on the instructor's computer. You have to copy them to your computers and change the IP addresses as shown on the table below.
To access the VMs you just need to map the shared folder. The instructor's IP address is 192.168.0.123 and the shared folder is named CEHv9.

Notes:

  • All VMs have 192.168.0.253 as their default gateway.

  • The Windows Server has two DNS servers: its own IP address and 8.8.8.8

  • The Windows 7 has two DNS servers: the Windows Server IP address and 8.8.8.8

  • Kali has 8.8.8.8 as its DNS server.

  • To preserve resources and have better performance, turn on only the VMs that you need for each exercise!
